Experiment 4 - Infinite Worlds

Hello!

Description

Creates an infinitely generated world with variation based on the world seed

Image inspiration

Technical

Utilizing p5.js, this experiment creates infite terraing which analyzes each tile one by one in order to determine modifications to it or its surrounding tiles. The terrain is generated using a seed, which determines either the color or shape of the tiles, or the background image.

Reflection

This experiment was a great way to learn about array manipulation in a program that updates every frame. There were many places where I needed to get creative in order to resolve issues with the program. For instance, something as seemingly simple as changing the colors of the surrounding tiles became a troublesome task because the draw function could alter any of the changes made later in the array before it became visible to the user.
